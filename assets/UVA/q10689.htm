Q10689: Yet another Number Sequence

讓我們根據以下的函數來定義一種數列：

f(0) = a


f(1) = b


f(n) = f(n-1) + f(n-2), n > 1

當 a=0 , b=1 時，這數列就是費伯那西數列（Fibonacci Sequence）。若改變 a 和 b 的值，你會得到許多不同的數列。在這個問題中，給你 a,b 的值，請你找出 f(n) 的最右邊m位數字。例如：a=0, b=1, f(12)=144，所以若 m=3，則答案是144。若 m=2，則答案是44。若 m=1，則答案是4。

Input

輸入的第一列有一個正整數代表以下有幾組測試資料（最多10000組）。每組測試資料一列。含有4個整數 a b n m。其中 a,b 在 [0,100] 的範圍內， n 在 [0,1000000000] 的範圍內，m 在 [1,4] 的範圍內。

Output

對每組測試資料輸出一列， f(n) 的最右邊m位數字。請不要輸出任何左方的 0（例如：089是不正確的）。
