Q10289: A Square & Equilateral Triangles

以下圖1,2,3,4,5,6,7說明了如何放置 1, 2, 3, 4, 5, 6, 8個相同大小的正三角形到一個正方形中的最佳方法。明顯的，如果正方形的大小均相同，放置在其中的正三角形的邊長一定是從圖1到圖7越來越小的。給你正方形的邊長，請你找出這7個圖中三角形的邊長各是多少。

你可以假設當圖形看起來與某個軸對稱時，就真的是對稱。

Input

每組測試資料一列含有1個浮點數S（0 <= S <= 10000），代表正方形的一邊長。

Output

對每組測試資料輸出一列，含有7個浮點數 t1, t2, t3, t4, t5, t6, t7。t1代表第1個圖中三角形的邊長，t2代表第2個圖中三角形的邊長，依此類推。請輸出到小數點後10位。本題有特殊的程式去判斷輸出，所以很小的準確度誤差會被忽略。

Sample input

0.0000001
0.0000002
0.0000003

Sample Output

0.0000001035 0.0000000816 0.0000000676 0.0000000634 0.0000000554 0.0000000526 0.0000000477
0.0000002071 0.0000001633 0.0000001353 0.0000001268 0.0000001109 0.0000001052 0.0000000953
0.0000003106 0.0000002449 0.0000002029 0.0000001902 0.0000001663 0.0000001577 0.0000001430
