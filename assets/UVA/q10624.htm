Q10624: Super Number

你有沒有認為 162456723 這個數很特別？ 如果你沒有發現他的特別的話，請看下圖（a | b 表示 b 可以被 a 整除）：

給你 n, m（0 < n < m < 30），請你找出一個 m 位數的正整數 X，使得對每一個 i（n <= i <= m），X 的前 i 位數是 i 的倍數。假如有不只這樣的一個 X 存在，請你輸出字典順序最小的一個。請注意：X 的第一位數（最左邊）不可以為 0。

Input

輸入的第一列有一個整數代表以下共有多少組測試資料（不會超過15組）。

每組測試資料一列。含有 2 個正整數  n 和 m。

Output

對每組測試資料輸出一列，這是第幾組測試資料及 X 。假如不存在 X，請輸出 -1。

輸出格式請參考Sample Output。

Sample Input Sample Output
3
1 10
3 29
27 26

Case 1: 1020005640
Case 2: -1
Case 3: 100000000000000000000000683
