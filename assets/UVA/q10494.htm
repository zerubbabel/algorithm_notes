Q10494: If We Were a Child Again

「嗚......如果再叫我做小學的數學，我敢保證這一次我一定不會出任何差錯。」一個聰明的大學生這麼說。

但是他的老師更聰明，他說：「好吧，我就指定一個這樣的計畫讓你去做，不要難過了。」

「真的嗎？」這個學生覺得很快樂，快樂到沒有發現老師臉上的微笑。

這個計畫是要這個學生做一個計算機來做基本的數學運算。但是像大多數的大學生一樣，他不想要自己做這個計畫。他只想要到處去收集程式來用。身為他的朋友，他要求你寫這樣的程式。聰明的你只同意幫他做整數的除法（division）和求餘數（mod）。

Input

輸入含有多組測試資料。

每組測試資料一列。含有2個正整數 m, n（  m 可能相當大， 0 < n < 231）及介於2數的一個運算符號（/ 或 % 分別代表整數除法及求餘數）。數及運算符號之間有空白，請參考Sample Input。

Output

對每組測試資料輸出1列，包含一個整數，為輸入運算的結果。請參考Sample Output。

110 / 100
98 % 10

2147483646 % 2147483647
